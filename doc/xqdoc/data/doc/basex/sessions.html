Library Module: data/doc/basex.xqm/sessions.xqm

URI: http://basex.org/modules/sessions
Description: This XQuery Module can only be called from users with Admin permissions. It contains functions for accessing and modifying all registered server-side sessions. This module is mainly useful in the context of Web Applications .
Author: BaseX Team
See: http://docs.basex.org/wiki/Module_Library

Functions

accessed($id)
Arguments:
$id xs:string
Returns:
xs:dateTime
Description: Returns the last access time of the session specified by $id .

close($id)
Arguments:
$id xs:string
Returns:
empty-sequence()
Description: Unregisters the session specified by $id .

created($id)
Arguments:
$id xs:string
Returns:
xs:dateTime
Description: Returns the creation time of the session specified by $id .

delete($id, $key)
Arguments:
$id xs:string
$key xs:string
Returns:
empty-sequence()
Description: Deletes a variable bound to the session specified by $id .

get($id, $key)
Arguments:
$id xs:string
$key xs:string
Returns:
xs:string?
Description: Returns the value of a variable bound to the session specified by $id . If the variable does not exist, an empty sequence or the optionally specified default value is returned instead.
Error: bxerr:BXSE0002 the value of a session variable could not be retrieved.

get($id, $key, $default)
Arguments:
$id xs:string
$key xs:string
$default xs:string
Returns:
xs:string
Description: Returns the value of a variable bound to the session specified by $id . If the variable does not exist, an empty sequence or the optionally specified default value is returned instead.
Error: bxerr:BXSE0002 the value of a session variable could not be retrieved.

ids()
Returns:
xs:string
Description: Returns the IDs of all registered sessions.

names($id)
Arguments:
$id xs:string
Returns:
xs:string*
Description: Returns the names of all variables bound to the session specified by $id .

set($id, $key, $value)
Arguments:
$id xs:string
$key xs:string
$value xs:string
Returns:
empty-sequence()
Description: Assigns a value to a variable bound to the session specified by $id .
Error: bxerr:BXSE0001 a function item was specified as value of a session variable.

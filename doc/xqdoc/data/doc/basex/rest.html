Library Module: data/doc/basex.xqm/rest.xqm

URI: http://exquery.org/ns/restxq
Description: This XQuery Module contains helper functions for the RESTXQ API, some of which are defined in the RESTXQ Draft .
Author: BaseX Team
See: http://docs.basex.org/wiki/Module_Library

Functions

base-uri()
Returns:
xs:anyURI
Description: This function returns the implementation defined base URI of the resource function.

uri()
Returns:
xs:anyURI
Description: This function returns the complete URI that addresses the Resource Function. This is the result of rest:base-uri appended with the path from the path annotation of the resource function.

wadl()
Returns:
element(wadl:application)
Description: This (unofficial) function returns a WADL description of all available REST services.

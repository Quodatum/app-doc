binary-to-bytes($bin)
Arguments:
$bin xs:anyAtomicType
Returns:
xs:byte*
Description: Returns the specified binary data (xs:base64Binary, xs:hexBinary) as a sequence of bytes.

binary-to-string($bytes)
Arguments:
$bytes xs:anyAtomicType
Returns:
xs:string
Description: Converts the specifed binary data (xs:base64Binary, xs:hexBinary) to a string. The UTF-8 default encoding can be overwritten with the optional $encoding argument.
Error: bxerr:BXCO0001 The input is an invalid XML string, or the wrong encoding has been specified. Invalid XML characters will be ignored if the CHECKSTRINGS option is turned off.
Error: bxerr:BXCO0002 The specified encoding is invalid or not supported.

bytes-to-base64($input)
Arguments:
$input xs:byte*
Returns:
xs:base64Binary
Description: Converts the specified byte sequence to a xs:base64Binary item. Conversion is cheap, as xs:base64Binary items are internally represented as byte arrays.
Error: bxerr:BXCO0001 The input cannot be represented in the specified encoding.
Error: bxerr:BXCO0002 The specified encoding is invalid or not supported.

integer-from-base($str, $base)
Arguments:
$str xs:string
$base xs:integer
Returns:
xs:integer
Description: Decodes an xs:integer from $str , assuming that it's encoded in base $base . The first $base elements of the sequence '0',..,'9','a',..,'z' are allowed as digits, case doesn't matter. Valid bases are 2 - 36. If $str contains more than 64 bits of information, the result is truncated arbitarily.

integer-to-base($num, $base)
Arguments:
$num xs:integer
$base xs:integer
Returns:
xs:string
Description: Converts $num to base $base , interpreting it as a 64-bit unsigned integer. The first $base elements of the sequence '0',..,'9','a',..,'z' are used as digits. Valid bases are 2, .., 36 .

string-to-base64($input)
Arguments:
$input xs:string
Returns:
xs:base64Binary
Description: Converts the specified string to a xs:base64Binary item. If the default encoding is chosen, conversion will be cheap, as both xs:string and xs:base64Binary items are internally represented as byte arrays. The UTF-8 default encoding can be overwritten with the optional $encoding argument.
Error: bxerr:BXCO0001 The input cannot be represented in the specified encoding.
Error: bxerr:BXCO0002 The specified encoding is invalid or not supported.

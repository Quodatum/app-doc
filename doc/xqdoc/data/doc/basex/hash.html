Author: BaseX Team

Functions

hash($value, $algorithm)
Arguments:
$value xs:anyAtomicType
$algorithm xs:string
Returns:
xs:base64Binary
Description: Computes the hash of the given $value , using the specified $algorithm . The specified values may be of type xs:string, xs:base64Binary, or xs:hexBinary. The following three algorihms are supported: MD5 , SHA-1 , and SHA-256 .
Error: bxerr:HASH0001 the specified hashing algorithm is unknown.
Error: bxerr:FORG0006 the specified value is neither a string nor a binary item.

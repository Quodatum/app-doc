Library Module: data/doc/basex.xqm/session.xqm

URI: http://basex.org/modules/session
Description: This XQuery Module contains functions for accessing and modifying server-side session information. This module is mainly useful in the context of Web Applications .
Author: BaseX Team
See: http://docs.basex.org/wiki/Module_Library

Functions

accessed()
Returns:
xs:dateTime
Description: Returns the last access time of a session.

close()
Returns:
empty-sequence()
Description: Unregisters a session and all data associated with it.

created()
Returns:
xs:dateTime
Description: Returns the creation time of a session.

delete($key)
Arguments:
$key xs:string
Returns:
empty-sequence()
Description: Deletes a session variable.

get($key)
Arguments:
$key xs:string
Returns:
item()*
Description: Returns the value of a variable bound to the current session. If the key is unknown, an empty sequence or the optionally specified default value is returned instead.
Error: bxerr:BXSE0002 the value of a session variable could not be retrieved.

get($key, $default)
Arguments:
$key xs:string
$default item()*
Returns:
item()*
Description: Returns the value of a variable bound to the current session. If the key is unknown, an empty sequence or the optionally specified default value is returned instead.
Error: bxerr:BXSE0002 the value of a session variable could not be retrieved.

id()
Returns:
xs:string
Description: Returns the session ID of a servlet request.

names()
Returns:
xs:string*
Description: Returns the names of all variables bound to the current session.

set($key, $value)
Arguments:
$key xs:string
$value item()*
Returns:
empty-sequence()
Description: Binds the specified key/value pair to a session.
Error: bxerr:BXSE0001 a function item was specified as value of a session variable.

sortui

Provides a UI to display and edit a sort preference.

Value is a name from a list with optional leading +/- ascending or descending flag.

The sortui directive provide several attributes:

  • ng-model: name of $scope member holding value {sort:"[+/-]field}
  • options: name of $scope member holding list of sortable names
